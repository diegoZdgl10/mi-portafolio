NÚMERO PALÍNDROMO

Descripción:

Dado un número entero x, devuelve true si x es un palindromo, y false en caso contrario.

Ejemplo 1

                        
                            /* Entrada */
                            x = 121
                        
                    
                        
                            /* Salida */
                            true

Ejemplo 2

                        
                            /* Entrada */
                            x = -121
                        
                    
                        
                            /* Salida */
                            false

Ejemplo 3

                        
                            /* Entrada */
                            x = 10
                        
                    
                        
                            /* Salida */
                            false

Solución:

                        
                            function esPalindromo(x) {
                                let xStr = x.toString()
                                let i = 0, j = xStr.length-1
                                while (i <= j) {
                                    if (xStr[i] !== xStr[j]) {
                                        return false
                                    }
                                    i++
                                    j--
                                }
                                return true
                            }

Complejidad

Tiempo:

\( O(n) \)

Espacio:

\( O(n) \)
